# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
GPU 냉각 효율 데이터 관리 프로그램
(GPU의 온도, 팬 속도, 전력 사용량을 입력받아 냉각 효율을 계산하고 통계적으로 분석하는 프로그램)

## 2. 핵심 기능 목록
1. GPU 이름, 온도, 팬 속도, 전력 사용량 입력  
2. 냉각 효율 = (팬 속도 ÷ 온도) 계산  
3. 평균 온도와 평균 효율 계산  
4. 최고 효율을 가진 GPU 정보 출력  
5. 온도가 일정 기준(예: 80℃ 이상) 초과 시 “⚠️ 과열 경고” 메시지 출력
## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열, 구조체를 설계)

**방법 1: 배열 사용 (기본)
```c
// GPU별 데이터 저장
char gpuNames[5][20];     // GPU 이름
float temperatures[5];    // 온도 (℃)
int fanSpeeds[5];         // 팬 속도 (RPM)
float powers[5];          // 전력 사용량 (W)
float efficiencies[5];    // 냉각 효율 (fanSpeed ÷ temperature)

int count = 0;            // 현재 GPU 개수
float avgTemp = 0.0;      // 평균 온도
float avgEff = 0.0;       // 평균 효율
int maxIndex = 0;         // 최고 효율 GPU의 인덱스

**방법 2: 구조체 사용 (선택)**
```c
// GPU 냉각 효율 정보를 구조체로 관리
struct Cooling {
    char gpuName[20];     // GPU 이름
    float temperature;     // 온도 (℃)
    int fanSpeed;          // 팬 속도 (RPM)
    float power;           // 전력 사용량 (W)
    float efficiency;      // 냉각 효율 (fanSpeed ÷ temperature)
};
